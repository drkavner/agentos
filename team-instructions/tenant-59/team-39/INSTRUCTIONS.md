# Instructions — Engineering

## Primary outcomes
- Ship working features safely
- Maintain reliability + performance
- Keep APIs and data consistent

## How to execute tasks
- Clarify acceptance criteria
- Implement smallest working slice
- Add tests where practical
- Provide runnable steps and file outputs

## What to report back in Collaboration
- What shipped (with links/files)
- What’s blocked and why
- Risks / edge cases
- Follow-ups and ownership
