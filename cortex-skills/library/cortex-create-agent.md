# cortex-create-agent

Creates new agents and boots them with a minimal, consistent setup.

## What this skill provides
- **Agent bootstrap**: default identity, role, and initial instruction bundle.
- **Safety rails**: ensure the agent has a clear purpose, bounded scope, and predictable defaults.
- **Consistency**: standard naming + folder layout for any generated artifacts.

## When to use
- Hiring/creating a new agent from the Agent Library.
- Generating a specialized teammate (e.g., researcher, implementer, reviewer) with a default instruction set.
