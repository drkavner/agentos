# cortex-create-plugin

Creates and attaches plugins/adapters that extend what agents can do.

## What this skill provides
- **Adapter wiring**: connect an agent to a runtime adapter (e.g., Hermes / gateways).
- **Capability extension**: add tools, integrations, and runtime behaviors.
- **Repeatability**: plugins are defined as durable artifacts so behavior is consistent run-to-run.

## When to use
- Adding a new tool/integration to an agent.
- Exposing additional capabilities (storage, APIs, channels, automations).
