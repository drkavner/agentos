# para-memory-files

Memory system skill based on PARA (Projects / Areas / Resources / Archive).

## What this skill provides
- A structured way to keep **durable context** across runs
- A convention for storing notes, decisions, and artifacts in predictable places
- A lightweight “working memory” workflow that avoids losing important facts

## When to use
- Long-running projects where agents need continuity
- Teams where multiple agents collaborate and must share context
